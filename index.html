РЕАЛИЗУЙТЕ ТРИГГЕР ПЕРЕКЛЮЧАТЕЛЬ

    Реализуйте триггер переключатель.
  1. При движении мышки с одной трети квадрата картинка изменяется на ту, в которой выделенно красным цветом первый пункт
  2. При движении мышки во второй трети квадрата картинка изменяется на ту, в которой выделенно красным цветом второй пункт
  3. При движении мышки во третьей трети квадрата картинка изменяется на ту, в которой выделенно красным цветом третий пункт
  4. При клике добавить переход по ссылке, для каждой трети квадрата своя ссылка
  5. Добавить адаптивную верстку
